# Weekly Content Plan

## Rotation Logic
Mon: Agency Reality
Tue: Marketing Truth
Wed: Brand Breakdown
Thu: Operator Insight
Fri: Personal Story / Win
Sat: Short punchy post (3–5 lines, one sharp observation)
Sun: Off — or light reply/engagement if something topical comes up

Mix formats: not every post needs to be long-form. Saturdays are deliberately short.
Never post the same bucket two days in a row.
